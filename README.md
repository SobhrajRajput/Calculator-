# Simple Calculator-
A modern and responsive Calculator application built using HTML, CSS, and JavaScript. This project provides real-time arithmetic calculations with a clean and user-friendly interface.

# 🚀 Features

User Interface: Clean and modern calculator design.

Real-Time Calculations: Perform instant arithmetic operations.

Responsive Design: Works smoothly on mobile and desktop devices.

Interactive Buttons: Smooth and user-friendly calculator controls.

Lightweight Project: Built using pure HTML, CSS, and JavaScript.

# 🛠️ Tech Stack
  HTML5

  CSS3

  JavaScript
